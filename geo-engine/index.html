Retro Game Development

The Worlds FIRST game engine dedicated to the development of retro and lower-end systems. Our engine targets system as old as Windows 7 and as new as Windows 11.

Built with Simplicity in Mind

Geo is built to be as simple of a process as possible! Built upon the DirectX11 Framework, it will work well on any computer from the lowest possible specs to the highest specs! The export process just a click of a button away!

Our Services

We provide a range of mainstream tools built into Geo to help with the development of your game.

Object Pooling

Geo is capable of rendering many on screen objects in a single frame. A stress test of the engine showed Geo could render 500,000 objects on screen while holding 20 FPS. Specs: Intel i3, 8GB DDR3, Integrated Graphics, Windows 7

Realtime Lighting

Although Geo isn't capable of the behemoth that is Unreals' Lumen, it still delivers realtime lighting solutions for realtime lighting.

Visual Scripting

Geo offers a Node-Based Scripting solution, Weave, to further simplify the game creation process. Users can drag and drop nodes from the Toolbox and connected to each other to create logic.
